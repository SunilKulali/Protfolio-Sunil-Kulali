# Sunil Kulali — Portfolio

![Portfolio banner](./public/banner-placeholder.png)

>A modern, responsive developer & data-analyst portfolio showcasing projects, experience, and an interactive resume-powered chatbot.

---

- 🔥 Live demo: https://SunilKulali.github.io/Protfolio-Sunil-Kulali (if enabled)
- ⚡ Tech: React 19 · Vite · Framer Motion · React Router
- 🖤 Theme: Black & Gold with subtle micro-interactions

---

## Highlights

- 7 polished pages: Home, About, Skills, Projects, Experience, Education, Contact
- Role switcher on `Skills` (Data Analyst, BA, DE, SQL)
- Bottom-right chatbot that answers questions from your resume data (`src/data/resumeData.js`)
- Fully responsive and keyboard-accessible

## Quick Start

Clone the repo and run locally:

```bash
git clone https://github.com/SunilKulali/Protfolio-Sunil-Kulali.git
cd Protfolio-Sunil-Kulali/portfolio
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

### Build

```bash
npm run build
npm run preview
```

---

## Tech Stack

- React 19
- Vite
- Framer Motion (animations)
- React Router

## Customize

- Update resume data: `src/data/resumeData.js`
- Update site text and pages in `src/pages/`
- Static assets: `public/` and `src/assets/`

---

## Deploy

To deploy to GitHub Pages from this repository root:

```bash
cd portfolio
npm run build
# push the `dist` from build to the gh-pages branch or enable GitHub Pages from repo settings
```

Tip: Use the `gh-pages` npm package or GitHub Actions for automated deploys.
